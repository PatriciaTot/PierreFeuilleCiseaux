Jeu de pierre feuille ciseaux

Règles du jeu

Le jeu de pierre feuille ciseaux est un jeu de main où deux joueurs choisissent chacun un symbole parmi pierre, feuille ou ciseaux. Le gagnant est déterminé par les règles suivantes :

  • La pierre bat les ciseaux (la pierre écrase les ciseaux)
  • La feuille bat la pierre (la feuille enveloppe la pierre)
  • Les ciseaux battent la feuille (les ciseaux coupent la feuille)
